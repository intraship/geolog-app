Welcome to GeoLog

This app records your boring information, saves your progress, and builds report/export files from what you enter.
What this app does

Every field you enter becomes part of the saved boring record. Sample data, remarks, casing, and photos are all stored so the app can generate review screens and exports.

Automatic progress

The app generates progress as you save each part of the workflow. Saved borings appear on the home screen, and exports are built from the information already entered.

Basic workflow

Start Report → save report info → create a boring → fill boring header → add samples/casing/photos → Review → Save & Export.

Important navigation

Use Back buttons to return to the previous screen. Use save buttons before leaving a step so your changes are kept in the boring record.

Field Log Report

How to use this app

1. Start ReportEnter the general report information first. This fills the defaults used when you create a new boring.
2. Fill the boringSave the boring header, then add samples, Shelby tubes, casing, photos, and remarks.
3. Review and exportUse Review to check the boring and create photo logs or other exports. Both original and cropped photo log options are available.
Everything you enter is part of the report record. Saving a step keeps your progress so you can come back later from the Saved Borings list.

Saved Borings

Boring workflow

  • Save the boring header first. The Point ID is the key field used to identify the boring in saved records and exports.
  • Then add samples, Shelby tubes, casing, patch/original photos, and remarks.
  • Use the buttons at the bottom of each section to save before moving on.
  • The app uses what you save here to build review pages, photo logs, and exports automatically.
Tip: going back is okay, but save the current step first so your latest edits are preserved.

Start Report

Fill out the report information for this job first. When you press Save Report Info, these values become the default information for new borings. Use New Boring to start entering field data.

What to know on this screen

  • This page stores project-level information like inspector, driller, rig, weather, and location.
  • Show Info reveals extra project details that also get saved into the report.
  • If you leave before saving, your newest changes on this screen may not be included in the next boring.

Boring workflow

  1. Fill out the Boring Header. The app automatically starts the boring time when you create a new boring.
  2. Check or edit Date, Time Started, and Time Finished. Time Finished is auto-generated when you go to Review, and you can edit it here or in Review.
  3. Press Edit Header to save header changes.
  4. Press Add Sample for regular samples. Press Add Shelby for undisturbed Shelby tube samples.
  5. Press Edit Casing to enter casing blows and depths when casing is needed.
  6. When the boring is complete, press Review to check everything, confirm times, and export.
Everything entered here is reported. Start time is created automatically for a new boring, and finish time is generated when you open Review. You can edit both before export.
